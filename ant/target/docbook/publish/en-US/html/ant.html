Chapter 1. INSTALACIÓN DEL APACHE ANT

  • Descargar el Ant: http://ant.apache.org/bindownload.cgi
  • *Directorio de descarga:*d:\download\ant
  • Directorio de Instalación: D:\javajdk\apache-ant
  • Configurar la variable de Entorno: ANT_HOME. Ejemplo: =D:\javajdk\apache-ant
  • Configurar el Java al Path del windows. Ejemplos: …; %ANT_HOME%\bin
  • Ejecutar la prueba de Instalación: ant - version
