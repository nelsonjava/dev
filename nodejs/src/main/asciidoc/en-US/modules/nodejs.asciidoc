== Iniciar el proyecto Node.js

[source, console]
----
   npm init
   npm init
   npm install --save-dev tape
   npm install --save-dev tap-spec
----

* --save-dev: Los módulos instalados lo va a savar como dev dependencia

* Los proyectos de node.js tienen dos tipos de dependencia: La dependencias del modulo como tal que va a utilizar en modo de producción o
  los dependencias de tipo de desarrollo que solo se utitilazan en el periodo de desarrollo.

* Los pruebas unitarias solo se hacen en el fase de desarrollo y por eso se instalando de tipo desarrollo.

== npm install --save-dev tape
